# Atlas

Atlas is a powerful, interactive visualization tool designed to explore the **Octra Network**. It provides a real-time, graphical representation of network entities, transactions, and relationships, making it easier to track funds and understand network activity.

<Card title="Live Beta" icon="rocket" href="https://atlas.0xio.xyz">
  Access the live beta version at **atlas.0xio.xyz**.
</Card>

## Key Features

- **Interactive Graph Visualization**: View addresses as nodes and transactions as links in a force-directed graph.
- **Real-Time Data**: Fetches live data from the Octra Network RPC and Octrascan API.
- **Wallet Integration**: Connect your **0xio Wallet** to authenticate and unlock advanced features.
- **Privacy Mode**: Toggle privacy to mask sensitive wallet addresses during demonstrations or public sharing.
- **Deep Analytics**: Inspect node details, balance, nonce, and transaction history directly from the graph interface.

## Access Control

Atlas is currently in **Beta** and limits access to authorized users.

1.  **Connect Wallet**: You must connect a 0xio Wallet to enter.
2.  **Access Code**: First-time users are required to enter a secure 5-digit access code (e.g., provided during the beta program).

## Getting Started

1.  Navigate to [atlas.0xio.xyz](https://atlas.0xio.xyz).
2.  Click "Connect Wallet" and approve the connection in your 0xio extension.
3.  Enter your beta access code.
4.  Start exploring by searching for an address (starts with `oct...`) or clicking on nodes in the graph.
